
Command Line Parameters
Arg1 = FPS
Arg 2 = ballSpeed

Example

breakout 30 5
//this is a breakout game with 30 fps and a ballSpeed of 5

The splashscreen is an image that is in the images folder. Therefore this should be run on the same directory that includes the images folder.

Use the mouse to move the paddle.

Left click on the splash screen to begin playing the game

Additional Features
Each level of bricks takes a different amount of hits to be destroyed. There is a random element where sometimes bricks will be hit multiple times depending on the luck of the user.
